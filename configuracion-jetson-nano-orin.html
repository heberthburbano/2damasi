Configuración Jetson nano orin

Vamos enseñar la configuración y las componentes que tienía la jetson.

En la flto de arriba podemos ver el dispositivo. se pude observar que el cable de alimentación es americano, y por lo que se comentó en clase salía más barato comprar el nuevo cable que el adapta

Si le damos la vuelta a la jetson veremos que tiene un disco nvme, que no viene con el dispositivo, sino que puso Pedro.

En la imagen anterior podemos observar que tenemos 3 puertos USC, 1 USB C, 1 displayPort, 1 rj45 y el puerto para la alimentación.

Para introducir el el sistema operativo lo hicimos mediante una tarjeta micro sd que tiene cargado un ubuntu especial para la jetson nano orin.

Le conectamos la alimentación, el displayport, ratón y teclado.

Con todo eso lo conectamos y encendimos, en nuestro caso tuvimos un error en el primer encendido y el SO se quedo cargando durante 15 mins, solo tuvimos que reiniciar y no tuvimos más problemas.

Le instalamos chrome y ya estaría.

Había un problemas con spnap, pero al actualizar el SO se solucionó.
