Crear repositorio y token en git hub

Para poder publicar la página en las git hub pages debemos:

  1. Crear un repositorio.
  2. Activar la git hub pages.
  3. Crear el token de git hub.
  4. Sincronizar publii con git hub.

Creación del repositorio

Lo primero que vamos a hacer es crear el repositorio de git hub con el nombre "2damasi".

Como recomendación para ahorrar paso luego, marca la opción "Add README".

Ahora vamos activar la git hub pages.

Para ellos vamos al apartado de configuración del repositorio y accedemos a la sección "pages".

La configuración debe ser como la que está en la captura.

Generar el token

Está parte es muy importante, ya que aquí sincronizar publii con git hub.

Para generar el token vamos a la configuración de perfil del usuario, en el final del la barra lateral izquierda veremos la opción "Developers option". y allí crearemos un token classic.

Marcar esa opción es importante porque sino publii no podrá realizar cambios en el repositorio.

En el menú de la parte izquierda de la pantalla principla de publii veremos un apartado que pone "server", al entrar en ese opción nos aparecerán varias opciones. La opción que nosotros debemos seleccionar es git hub.

Como puedes ver en la imagen hemos indicado que las rutas sean relativas para evitar poblemas luego con las imagenes, tema y enlaces.
